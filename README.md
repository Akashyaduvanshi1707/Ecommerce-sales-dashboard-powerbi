# Ecommerce Sales Dashboard - Power BI

## Project Overview
Interactive Power BI dashboard created to analyze ecommerce sales performance, customer purchasing behavior, profit trends, and category-wise business insights.

The dashboard helps track KPIs, sales trends, quantity sold, and profit distribution using visual analytics.

---

## Tools Used
- Power BI
- Excel
- Power Query

---

## Key Insights
- Sales performance analysis
- Profit by month analysis
- Category-wise quantity insights
- Payment mode analysis
- Customer purchase trends
- Sub-category profit analysis

---

## Features
- Interactive filters and slicers
- KPI cards for quick insights
- Dynamic charts and visualizations
- Business performance tracking dashboard
